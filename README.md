# 🛠️ API REST para Inventario de Productos (Backend)

Este repositorio contiene el backend de una aplicación web de inventario. Construido con **Node.js**, **Express** y **SQLite**, proporciona una API REST para gestionar productos (crear, listar, editar, eliminar).

🌐 **Backend en vivo (Render):**  
https://inventario-backend-4i4e.onrender.com

📦 Frontend conectado (GitHub Pages):  
https://juancm0.github.io/Inventario-frontend/

---

## 🧩 Endpoints disponibles

| Método | Endpoint           | Descripción                   |
|--------|--------------------|-------------------------------|
| GET    | `/productos`       | Listar todos los productos    |
| POST   | `/productos`       | Agregar un nuevo producto     |
| PUT    | `/productos/:id`   | Editar un producto existente  |
| DELETE | `/productos/:id`   | Eliminar un producto por ID   |
